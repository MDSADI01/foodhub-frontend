# 🚀 Foodyverse Frontend

Foodyverse is a modern and responsive food ordering frontend application built with React and Next.js. It provides a smooth and user-friendly interface for browsing food items, viewing details, and placing orders with an intuitive shopping experience.

In many local food ordering systems, users face slow, outdated, and confusing interfaces that make browsing food items and placing orders difficult. Most platforms lack a smooth user experience, responsive design, and real-time interaction between menu browsing and ordering.

Foodyverse solves this problem by providing a fast, modern, and user-friendly food ordering frontend that simplifies the entire browsing and ordering process.

The application is designed with a clean UI, responsive layout, and optimized performance to ensure a seamless experience across all devices. It communicates with a backend API to fetch dynamic food data and manage user interactions efficiently.

Foodyverse focuses on delivering a fast, scalable, and visually appealing frontend for a complete food ordering system.

---

## 🛠️ Tech Stack

- Next.js  
- React  
- TypeScript  
- Tailwind CSS
- BetterAuth
- REST API Integration  

---

## ✨ Key Features

- 🍔 Browse food items with categories  
- 📄 View detailed food descriptions and images  
- 🛒 Add items to cart and manage orders  
- ⚡ Fast and optimized performance with Next.js  
- 📱 Fully responsive design for all devices  
- 🔗 API integration for dynamic food data  
- 🔐 Secure authentication (Admin, Provider, Customer roles)

### Setup Instructions

# Clone the repository
git clone https://github.com/MDSADI01/foodhub-frontend.git

# Navigate to project folder
cd foodyverse-frontend

# Install dependencies
npm install

# Run development server
npm run dev


# Env Variables:
NEXT_PUBLIC_API_URL=https://foodyverse-backend.vercel.app/
